# SimpleChatBot

運用DialogFlow製作出一個簡易的餐廳預約聊天機器人

說明：

1. 可向聊天機器人打招呼

2. 可詢問餐廳位置或營業時間

3. 可進行座位預定

4. 告訴機器人要預約的位子數

5. 再告知時間就完成啦~~
